# MediQueue : AI-Powered Hospital Queue Management System

## 🚀 Tech Stack
- Java 17+
- Spring Boot
- Spring Cloud (Eureka, Gateway)
- JWT Authentication

## 🧩 Architecture
- Discovery Service (Eureka)
- API Gateway (Spring Cloud Gateway)
- Patient Service (upcoming)

## 🔐 Features
- Centralized authentication using Gateway filter
- JWT-based security
- Service discovery

## ▶️ How to Run
1. Start discovery-service (8761)
2. Start api-gateway (8080)
3. Start other services
